API Basejump: Timestamp microservice

User stories:

  1. I can use a string as a aramter, and it will check to see whether that string either contains a unix timestamp or a natural language date (example: January 1, 2016)
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp, it return null for those properties.

Example usage:

https://timestamp-tanya53.c9users.io/December%2015,%20215

https://timestamp-tanya53.c9users.io/1450137600

Example output:

{"unix":1450137600, "natural":"December 15, 2015"}
